# Library Management System

## Project Description
This project is a Library Management System developed using Java and Object-Oriented Programming (OOP) principles.
The system allows managing books, members, and borrowing operations in a structured manner.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- UML Class Diagram
- GitHub Projects (Kanban)

## System Features
- Search books by title and author
- Borrow available books
- Return borrowed books
- Track book availability
- Manage library members and loans

## UML Class Diagram
The system design is represented using a UML class diagram.

> UML diagram files are included in the repository.

## Main Classes
- **Book**: Represents a book in the library
- **Member**: Represents a library member
- **StudentMember**: Extends Member with student-specific properties
- **Loan**: Handles book borrowing operations
- **LibraryManager**: Manages books, members, and loans

## Project Management
This project follows a Kanban-based workflow using GitHub Projects.
